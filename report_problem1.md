# 问题一：基于方向场估计的条纹图像去噪

## 1. 问题分析

### 1.1 问题背景

条纹图案（Fringe Pattern）广泛存在于光学干涉测量、表面轮廓检测等精密工程领域。实际采集过程中，图像不可避免地被噪声污染，严重影响后续相位提取和方向场估计的精度。问题一要求对 10 幅含噪条纹图像进行去噪处理，在去除噪声的同时尽可能保留条纹结构的方向信息。

### 1.2 噪声的物理本质

含噪图像中可能存在两种物理本质截然不同的噪声：

| 噪声类型 | 数学模型 | 统计特征 | 物理来源 |
|---------|---------|---------|---------|
| 加性高斯噪声 | I = I₀ + n，n ~ N(0, σ²) | Var ≈ σ²（常数） | 传感器热噪声、电子学噪声 |
| 乘性散斑噪声 | I = I₀ · n，n > 0 | Var ≈ σₛ² · μ²（方差与均值平方成正比） | 相干光源干涉散斑 |

两者在统计特性上"水火不容"：加性噪声的局部方差与均值无关，乘性噪声的局部方差与均值的平方强正相关。**强行用同一套参数处理两种噪声，必然顾此失彼。** 因此，自动鉴别噪声类型并选择对应处理策略是解决问题的第一步。

### 1.3 评价指标

采用**平均绝对误差（MAE）**作为评价指标：

```
MAE = (1/N) Σᵢ |fᵢ - gᵢ|
```

其中 f 为去噪图像，g 为理想图像，像素值域为 [0, 255]（uint8）。MAE 越低越好。

## 2. 数学模型建立

### 2.1 噪声类型物理感知器

在宏观无重叠图块（32×32）上计算局部均值 μ 和局部方差 Var，通过以下统计量自动鉴别噪声类型：

**Pearson 相关系数**：计算 Var 与 μ² 的线性关联度：

```
r = cov(Var, μ²) / (σ_Var · σ_μ²)
```

**线性回归**：拟合 Var = a + b · μ²，斜率 b 为等效乘性噪声系数。

**判别准则**：
- r > 0.3（阈值）→ 乘性散斑噪声
- r ≤ 0.3 → 加性高斯噪声

实验验证：对高斯噪声样例（image1），r = 0.043；对散斑噪声样例（image2），r = 0.992。两类噪声被清晰分离。

### 2.2 结构张量与方向场估计

结构张量（Structure Tensor）是方向场估计的核心工具，用于提取图像的局部方向信息。

**梯度计算**（高斯预平滑尺度 σ）：

```
Ix = Gσ * ∂u/∂x，  Iy = Gσ * ∂u/∂y
```

**结构张量**（积分尺度 ρ）：

```
         ⎡ Ix²    Ix·Iy ⎤       ⎡ J₁₁  J₁₂ ⎤
Jρ = Gρ *⎢               ⎥  =    ⎢            ⎥
         ⎣ Ix·Iy  Iy²   ⎦       ⎣ J₁₂  J₂₂ ⎦
```

其中 Gσ 和 Gρ 为高斯核，σ 控制噪声尺度（梯度预平滑），ρ 控制积分尺度（结构张量平滑）。

**特征值分解**：

```
μ₁ = (J₁₁ + J₂₂ + √[(J₁₁ - J₂₂)² + 4·J₁₂²]) / 2
μ₂ = (J₁₁ + J₂₂ - √[(J₁₁ - J₂₂)² + 4·J₁₂²]) / 2
```

μ₁ ≫ μ₂ 表示局部存在强方向性结构（条纹），μ₁ ≈ μ₂ 表示均匀区域或纯噪声。

### 2.3 Weickert 相干性动态扩散张量

定义相干性度量（结构张量的不变量）：

```
k = (J₁₁ - J₂₂)² + 4·J₁₂²
```

k 正比于 (μ₁ - μ₂)²，衡量局部各向异性程度。基于 k 构建动态特征值：

```
λ₁ = α

λ₂ = α + (1 - α) · exp(-C / (k + ε))
```

其中 α 为背景扩散率（最小扩散强度），C 为相干性灵敏度参数。

**物理解释**：
- **强噪声区**（k ≈ 0）：λ₂ → α，两个特征值近似相等 → **各向同性热传导**，均匀平滑不产生伪结构
- **条纹清晰区**（k ≫ 0）：λ₂ → 1，λ₁ = α ≪ 1 → **高度非等向扩散**，沿条纹切线方向强平滑、法线方向弱扩散

扩散张量 D 由特征值和特征向量重构：

```
D = λ₁·v₁·v₁ᵀ + λ₂·v₂·v₂ᵀ
```

### 2.4 各向异性扩散 PDE

CED 求解以下抛物型 PDE：

```
∂u/∂t = div(D · ∇u)
```

**离散格式**：守恒型半点通量差分，以 x 方向右侧半点通量为例：

```
F(i+½, j)ˣ = D₁₁ᴿ · δₓ⁺u + D₁₂ᴿ · δ̄ᵧu
```

其中 D₁₁ᴿ = ½(D₁₁ⁱʲ + D₁₁ⁱʲ⁺¹) 为半点处扩散系数的算术平均，δₓ⁺ 为前向差分，δ̄ᵧ 为跨半点的中心差分平均。

**离散散度**：

```
div(D·∇u) ≈ (Fᴿˣ - Fᴸˣ) + (Fˢʸ - Fᴺʸ)
```

**时间积分**：显式 Euler 格式 uⁿ⁺¹ = uⁿ + Δt · div(D·∇uⁿ)。

**CFL 稳定性条件**：Δt ≤ 1/(4·D_max)，其中 D_max = max(λ₂)。当 λ₂ ≤ 1 时，Δt = 0.2 满足稳定条件。

**边界条件**：Neumann 零通量条件 ∂u/∂n|_∂Ω = 0，通过边缘扩展填充（edge padding）实现。

### 2.5 同态映射（散斑路径专用）

乘性散斑噪声在对数域转化为加性噪声：

```
log(I) = log(I₀) + log(n)
```

取 u = log(I + ε)（ε = 10⁻⁶ 防止 log 0），在对数域执行 CED 后通过逆映射恢复：

```
Î = exp(u_denoised) · M_prior / M_out
```

其中乘法均值校正因子利用条纹图的物理先验：干净条纹图案 I₀ = ½(1 + cosφ) 的均值为 E[I₀] = 0.5。

### 2.6 Lee 滤波器预前处理

Lee 滤波器是乘性散斑噪声的最小均方误差（MMSE）估计器：

```
Î₀ = Ī + k · (I - Ī)
```

其中自适应权重：

```
k = clip((σᴵ² - σₙ²) / (σᴵ² + ε), 0, 1)
```

- 均匀区域（σᴵ² ≈ σₙ²）：k → 0，输出为局部均值（最大平滑）
- 高方差区域（边缘、纹理）：k → 1，输出保留原始像素（保边）

## 3. 算法流程

```
输入：含噪图像 I ∈ [0,1]
│
├── Step 0: 噪声类型物理感知
│   └── 计算 corr(Var, μ²) → 判定加性/乘性
│
├── [乘性散斑路径]
│   ├── Step 1a: Lee 滤波预前处理 → I_pre
│   ├── Step 1b: 高斯预平滑(σ=1.5) → I_smooth
│   ├── Step 2:  同态映射 u = log(I_pre + ε)
│   ├── Step 3:  在 log(I_smooth) 上计算结构张量 Jρ
│   ├── Step 4:  构建 Weickert 扩散张量 D（固定，不随迭代更新）
│   ├── Step 5:  迭代求解 PDE: uⁿ⁺¹ = uⁿ + Δt · div(D·∇uⁿ)
│   ├── Step 6:  逆映射 u = exp(u)
│   └── Step 7:  乘法均值校正 u ← u × 0.5 / mean(u)
│
├── [加性高斯路径]
│   ├── Step 1:  直接使用 I（无同态映射）
│   ├── Step 2:  每步迭代重新计算结构张量 + 扩散张量
│   └── Step 3:  迭代求解 PDE
│
└── 输出：clip(u, 0, 1)
```

**双态分流设计差异**：

| 设计维度 | 散斑路径 | 高斯路径 |
|---------|---------|---------|
| 同态映射 | log(I + ε) → exp(u) | 关闭 |
| Lee 预滤波 | 开启（7×7 窗口） | 关闭 |
| 方向场预平滑 | 高斯 σ = 1.5 | 无需 |
| 扩散张量策略 | **固定**（预平滑后计算一次） | **自适应**（每步迭代更新） |
| 积分尺度 ρ | ≥ 8（大尺度压制散斑） | ≤ 2（保留精细条纹） |
| 迭代次数 | 100 | 50 |
| 均值校正 | 乘法（先验 M = 0.5） | 加法（保持输入均值） |

## 4. 关键参数设置

### 4.1 参数表

| 参数 | 符号 | 散斑路径 | 高斯路径 | 物理含义 |
|------|------|---------|---------|---------|
| 梯度预平滑尺度 | σ | 1.0 | 1.0 | 梯度估计的抗噪性 |
| 结构张量积分尺度 | ρ | 8.0 | 2.0 | 方向场估计的空间平均范围 |
| 迭代次数 | N | 100 | 50 | 扩散的时间演化长度 |
| 时间步长 | Δt | 0.2 | 0.25 | 显式 Euler 步长 |
| 背景扩散率 | α | 0.01 | 0.01 | 噪声区最小扩散强度 |
| 相干性灵敏度 | C | 10⁻⁶ | 10⁻⁸ | λ₂ 从各向同性到非等向的过渡锐度 |
| 预平滑尺度 | σ_pre | 1.5 | — | 方向场预平滑强度 |
| 条纹均值先验 | M_prior | 0.5 | — | 干净条纹的期望均值 |

### 4.2 参数选择依据

散斑路径参数通过**网格搜索**确定，搜索空间与最优结果：

| 参数 | 搜索范围 | 最优值 | MAE 变化幅度 |

### 5.2 消融实验

逐步叠加各优化模块，验证每项改进的独立贡献（以 image2 散斑样例为测试对象）：

| 阶段 | 配置 | MAE | SSIM | 累计改善 |
|------|------|-----|------|---------|
| A. v4 基线 | log(1+I)，加法均值平移，ρ=5，50 iter | 22.63 | — | — |
| B. + 乘法均值校正 | 改用 u × 0.5 / ū | 8.24 | 0.898 | **−63.6%** |
| C. + Lee 预滤波 | 7×7 Lee 滤波 + ρ=8，100 iter | 7.48 | 0.914 | −9.2% |
| D. + 预平滑方向场 | 高斯 σ=1.5 预平滑结构张量 | 7.44 | 0.916 | −0.5% |
| E. + 更多迭代 | 150 iter（渐进改善） | 7.13 | — | −4.2% |

**核心发现**：
1. **乘法均值校正**是最大单一改进（MAE 从 22.63 降至 8.24），贡献了总改善的 **87%**
2. **Lee 预滤波**提供第二大幅改善（8.24 → 7.48），使 PDE 求解器的输入更干净
3. **预平滑方向场**进一步精细化（7.48 → 7.44），避免 CED 沿噪声伪结构扩散
4. 迭代次数从 100 增至 150 可进一步降至 MAE=7.13，但边际收益递减

## 6. 方法局限性讨论

### 6.1 条纹均值先验假设

乘法均值校正依赖先验知识 E[I₀] = 0.5，该假设对于标准归一化条纹 I₀ = ½(1 + cosφ) 成立。但若图像未按此形式归一化（如存在 DC 偏移或非对称相位分布），该先验将引入系统性偏差。

**改进方向**：从图像自身统计量（如噪声感知器估计的散斑参数 σₛ）推导校正因子，减少对外部先验的依赖。

### 6.2 固定扩散张量的权衡

散斑路径采用固定扩散张量（从预平滑图像计算一次），优势是避免噪声对方向场的持续干扰，代价是丧失了对扩散过程的自适应性——随着去噪进行，方向场本应越来越准确，但固定张量无法利用这一信息。

**改进方向**：两阶段策略——第一阶段用固定张量获得初步结果，第二阶段用初步结果重新估计方向场后再执行若干步自适应 CED。

### 6.3 极端噪声下的退化行为

当信噪比极低（散斑因子 σₛ > 0.5）时，Lee 预滤波和方向场预平滑均可能无法提取可靠的方向信息，CED 将退化为近似各向同性扩散，导致条纹边缘模糊。

### 6.4 泛化性限制

本方法针对条纹图案设计，利用了条纹的相干性和均值先验。对于非条纹结构图像（如自然图像、医学图像），方向场估计的可靠性下降，乘法均值校正的先验不再成立，需要重新设计参数和校正策略。

### 6.5 计算效率

高斯路径每步迭代需重新计算结构张量和扩散张量（复杂度 O(HWρ²)），对 256×256 图像约 4 秒/50 步。散斑路径因固定张量仅需单次结构张量计算，80~100 步迭代几乎实时完成。对于更大分辨率图像，需考虑半隐式格式或多尺度加速策略。
